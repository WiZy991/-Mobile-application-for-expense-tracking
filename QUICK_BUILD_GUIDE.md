# 🚀 Быстрая сборка Android приложения

## ✅ Ваше приложение готово к сборке!

Проект полностью настроен для создания APK файла, который можно установить на Android телефон.

## 📱 Шаг 1: Откройте проект в Android Studio

1. Запустите Android Studio
2. File → Open
3. Выберите папку проекта: `-Mobile-application-for-expense-tracking`
4. Дождитесь синхронизации Gradle (может занять несколько минут)

## 🔨 Шаг 2: Соберите APK

### Способ A: Debug APK (для тестирования)

1. В Android Studio: **Build → Build Bundle(s) / APK(s) → Build APK(s)**
2. Дождитесь завершения сборки
3. APK будет в: `app/build/outputs/apk/debug/app-debug.apk`

### Способ B: Через командную строку

Откройте терминал в корне проекта и выполните:

**Windows:**
```powershell
.\gradlew.bat assembleDebug
```

**Mac/Linux:**
```bash
./gradlew assembleDebug
```

## 📲 Шаг 3: Установите на телефон

### Вариант 1: Через USB (ADB)
```bash
adb install app/build/outputs/apk/debug/app-debug.apk
```

### Вариант 2: Вручную
1. Скопируйте `app-debug.apk` на телефон
2. На телефоне: Настройки → Безопасность → Неизвестные источники (включить)
3. Откройте APK файл на телефоне и установите

## ⚙️ Шаг 4: Настройте URL для реального устройства

**ВАЖНО!** Для работы на реальном телефоне нужно изменить URL API.

1. Откройте файл: `app/src/main/java/com/example/worldcashbox/data/api/ApiConfig.kt`
2. Найдите строку: `private const val DEFAULT_DEVICE_URL = "http://192.168.1.100:3000/api/"`
3. Замените `192.168.1.100` на IP адрес вашего компьютера

**Как узнать IP адрес:**
- Windows: откройте командную строку и введите `ipconfig`
- Mac/Linux: откройте терминал и введите `ifconfig`
- Найдите IPv4 адрес (например: 192.168.1.100)

4. Пересоберите APK после изменения

## ✅ Готово!

Теперь у вас есть APK файл, который можно установить на любой Android телефон (версия 8.1 и выше).

## 🐛 Если что-то не работает

1. **Ошибки компиляции:** Build → Clean Project → Rebuild Project
2. **Не подключается к API:** Проверьте IP адрес и что backend запущен
3. **APK не устанавливается:** Включите "Неизвестные источники" в настройках Android
