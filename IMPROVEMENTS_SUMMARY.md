### 🐛 **Исправленные баги**

1. ✅ Ошибка `.toFixed() is not a function`

   - Проблема: PostgreSQL возвращал DECIMAL как строки
   - Решение: добавлен `Number()` перед `.toFixed()`
   - Файлы: DashboardScreen, HistoryScreen, ServicesScreen, AnalyticsScreen, BalanceScreen

2. ✅ Неправильный импорт локали

   - Было: `import { ru } from 'date-fns/locale/ru'`
   - Стало: `import ru from 'date-fns/locale/ru'`
   - Файлы: все экраны с датами

3. ✅ Отсутствующий favicon.png
   - Убрана секция `web` из `app.json`
   - (не нужна для мобильного приложения)

## 💡 Как использовать новые функции

### Быстрый старт:

1. Откройте приложение
2. Посмотрите новый дизайн главного экрана
3. Нажмите на аватар вверху справа → откроется профиль
4. Попробуйте быстрые действия (4 карточки)
5. Перейдите в Настройки (последняя вкладка внизу)

### Редактирование профиля:

1. Профиль → Редактировать
2. Измените данные
3. Сохранить

### Настройка уведомлений:

1. Настройки → Уведомления
2. Включите/выключите переключатели
