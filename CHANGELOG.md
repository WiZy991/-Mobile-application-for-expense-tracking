# История изменений

## [1.1.0] - 2025-11-20

### ✨ Добавлено

#### Новые экраны

- **ProfileScreen** - полноценный экран профиля пользователя

  - Редактирование имени, email, телефона
  - Отображение баланса с цветовой индикацией
  - Статистика использования
  - Информация о регистрации и СБИС
  - Выход из аккаунта с подтверждением

- **SettingsScreen** - комплексный экран настроек
  - Управление аккаунтом
  - Настройки уведомлений (Push, Email)
  - Настройки приложения (Темная тема, Биометрия)
  - Управление данными (Очистка кэша, Экспорт)
  - Поддержка и информация

#### Функционал

- Красивая навигация между экранами
- Кнопка профиля в шапке главного экрана
- Быстрые действия на главном экране (4 карточки)
- Цветовая индикация баланса (зеленый/красный)
- Подтверждения для важных действий
- Индикаторы загрузки

### 🎨 Улучшения дизайна

#### DashboardScreen

- Новый градиентный заголовок
- Кнопка профиля с аватаром пользователя
- Улучшенная карточка баланса
- Кнопка "Пополнить"
- Раздел "Быстрые действия" с 4 карточками
- Иконки и цветовая кодировка

#### Навигация

- Обновленная нижняя панель (5 вкладок)
- Главная, История, Аналитика, Уведомления, Настройки
- Убраны дубликаты (Баланс, Услуги доступны через быстрые действия)

### 🐛 Исправления

- Исправлена ошибка с `.toFixed()` для строковых значений из БД
- Все числовые значения теперь корректно преобразуются через `Number()`
- Исправлен импорт локали `ru` из `date-fns`
- Убран отсутствующий `favicon.png` из конфигурации

### 📝 Документация

- Добавлен FEATURES.md с описанием новых функций
- Добавлен CHANGELOG.md
- Обновлен README.md

## [1.0.0] - 2025-11-20

### Первый релиз

#### Backend

- REST API на Node.js + Express
- Аутентификация через JWT
- База данных PostgreSQL
- Интеграция со СБИС
- Система уведомлений (Email, Telegram)
- Фоновые задачи (синхронизация, напоминания)

#### Mobile App

- React Native + Expo
- Авторизация и регистрация
- Дашборд с балансом
- История транзакций
- Аналитика расходов
- Управление услугами
- Уведомления

#### API Endpoints

- `/api/auth` - аутентификация
- `/api/clients` - управление клиентами
- `/api/payments` - транзакции
- `/api/services` - услуги
- `/api/analytics` - аналитика
- `/api/sbis` - интеграция СБИС
- `/api/notifications` - уведомления
