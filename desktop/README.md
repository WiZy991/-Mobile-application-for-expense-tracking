# WorldCashBox Staff (ПК)

Десктопное приложение для **инженеров, менеджеров и поддержки**: тикеты, чаты, аналитика. Работает **только онлайн** через ваш API (`…/api/`).

## Запуск в разработке

```bash
cd desktop
npm install
npm start
```

При первом входе укажите URL API, например: `http://ВАШ-СЕРВЕР:3000/api/` или продакшен-адрес с `/api` в конце.

## Сборка установщика Windows (.exe)

```bash
cd desktop
npm install
npm run build:win
```

Готовый установщик: папка `desktop/dist/` (файл вида `WorldCashBox Staff Setup x.x.x.exe`).

Требуется установленный Node.js и интернет (при первой сборке скачивается Electron).

## Роли

- **Инженер / support:** мои или все тикеты, ответы, смена статуса, «взять в работу», чаты, аналитика.
- **Менеджер:** все тикеты (только просмотр и чаты по правам API), аналитика по всем.

Логика совпадает с мобильным приложением и бэкендом `staff` / `conversations`.
