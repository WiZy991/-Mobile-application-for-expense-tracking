
## 🐛 Если что-то не работает

### Ошибка: "Ошибка подключения к СБИС"

**Проверьте** `mobile/src/config/sbisConfig.js`:

```javascript
login: 'tenditnika',      // ✅ Правильный?
password: 'Tenditnik1!',  // ✅ Правильный?
```

---

### Ошибка: "Network request failed"

**Проверьте** `mobile/app.json`:

```json
"extra": {
  "apiUrl": "http://192.168.1.100:3000/api"  // ✅ Ваш IP?
}
```

**Как узнать IP:**

```bash
# Windows
ipconfig

# Mac/Linux
ifconfig
```

---

### Ошибка: "Клиент не найден в CRM"

**Это нормально!**

- Значит контрагента нет в CRM
- Приложение попробует найти в ЕГРЮЛ
- Или покажет базовую информацию
