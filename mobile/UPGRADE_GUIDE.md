### 👤 Новый экран профиля

Теперь вы можете:

- Просматривать все свои данные

- Видеть статистику использования

**Как открыть:**

- Нажмите на круглую кнопку с вашей инициалой в правом верхнем углу главного экрана
- Или через: Настройки → Профиль

### 🎯 Быстрые действия

На главном экране теперь есть 4 красивые карточки:

- 💰 **Баланс** (зеленая) - посмотреть детали баланса

**Просто нажмите** на любую карточку!

## 🚀 Как начать пользоваться

### Первый запуск после обновления

1. **Обновите данные:**

   - Потяните главный экран вниз для обновления

2. **Настройте профиль:**

   - Нажмите на аватар вверху
   - Проверьте свои данные
   - При необходимости отредактируйте

3. **Настройте уведомления:**

   - Перейдите в Настройки
   - Включите нужные типы уведомлений

4. **Изучите быстрые действия:**
   - Попробуйте нажать на каждую карточку

## 💡 Полезные советы

### Навигация

- **Главный экран**: свайп вниз = обновление данных
- **Профиль**: нажмите на аватар (круглая кнопка вверху справа)
- **Быстрые действия**: самый быстрый способ попасть в нужный раздел

### Цветовая индикация

- 🟢 Зеленый баланс = всё хорошо
- 🔴 Красный баланс = есть задолженность
- 🟠 Оранжевая плашка = непрочитанные уведомления

### Редактирование профиля

1. Откройте профиль
2. Нажмите "Редактировать"
3. Измените данные
4. Нажмите "Сохранить"

### Выход из аккаунта

- Настройки → Выйти из аккаунта
- Или: Профиль → Выйти из аккаунта

## 🐛 Что-то не работает?

### Приложение не обновляется?

1. Полностью закройте приложение
2. Откройте заново
3. Потяните экран вниз для обновления

### Не видно новых функций?

1. Убедитесь, что backend обновлен
2. Проверьте подключение к интернету
3. Перезапустите приложение

### Баланс отображается неправильно?

- Это могут быть старые данные
- Потяните экран вниз для обновления

### Не можете войти в профиль?

- Проверьте, что вы авторизованы
- Попробуйте выйти и войти заново
