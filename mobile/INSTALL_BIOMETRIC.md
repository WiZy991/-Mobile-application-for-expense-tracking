# Установка expo-local-authentication

Для работы биометрии необходимо установить пакет `expo-local-authentication`.

## Установка

```bash
cd mobile
npm install expo-local-authentication
```

Или если используете yarn:

```bash
cd mobile
yarn add expo-local-authentication
```

## Примечание

Биометрия работает только на мобильных устройствах (iOS и Android). На веб-платформе функция автоматически отключена и не вызывает ошибок.

## После установки

Перезапустите приложение:

```bash
npm start
```
