# Объединение локальной и удалённой истории

## Проблема
На GitHub уже есть коммиты, которых нет локально. Нужно объединить истории.

## Решение

Выполните эти команды по порядку:

```powershell
# 1. Получить изменения с GitHub и объединить истории
git pull origin main --allow-unrelated-histories

# 2. Если Git попросит ввести сообщение коммита для merge, 
#    просто закройте редактор (если откроется) или нажмите Enter

# 3. После успешного объединения, загрузите на GitHub
git push -u origin main
```

## Если будут конфликты при merge

Если Git покажет конфликты:

1. Откройте файлы с конфликтами
2. Найдите строки с `<<<<<<<`, `=======`, `>>>>>>>`
3. Оставьте нужный код, удалите маркеры конфликтов
4. Затем выполните:

```powershell
# Добавить разрешённые конфликты
git add .

# Завершить merge
git commit -m "Merge remote and local history"

# Загрузить на GitHub
git push -u origin main
```

## Альтернатива: перезаписать GitHub (если не важно, что там было)

⚠️ **ВНИМАНИЕ:** Это удалит всё, что было на GitHub!

```powershell
git push -u origin main --force
```

Используйте это только если вы уверены, что хотите удалить старые коммиты на GitHub.
