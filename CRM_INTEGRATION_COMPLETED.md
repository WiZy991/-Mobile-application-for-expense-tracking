## 🔧 Что требуется для полноценной работы

### 1. OAuth Credentials

В файле `backend/src/routes/sbisProxy.js` замените:

```javascript
const SBIS_APP_CLIENT_ID = '5711216329531646';  // ← ВАШИ ДАННЫЕ
const SBIS_APP_SECRET = 'LZTMTNJ5F4ZJKUMSTILYTZVW';  // ← ВАШИ ДАННЫЕ
```

**Как получить:**
1. Зайдите на https://online.sbis.ru
2. Настройки → Интеграция → API
3. Создайте приложение
4. Скопируйте `app_client_id` и `app_secret`

### 2. Тарифы СБИС

Для работы CRM API нужны активные тарифы:
- ✅ **СБИС CRM** - для работы с контрагентами и сделками
- ✅ **API доступ** - для программного доступа

### 3. Права доступа

OAuth приложение должно иметь права на:
- Чтение списка контрагентов
- Чтение сделок CRM
- Чтение документов

## 🎓 Техническая информация

### Архитектура решения:

```
Mobile App (React Native)
    ↓
    sbisGetClientFromCRM()
    ↓
Backend API (/crm-client-oauth)
    ↓
    ├─ Автоматическая OAuth авторизация
    │  (app_client_id + app_secret)
    ↓
    ├─ Поиск контрагента через:
    │  ├─ Навигация.Контрагент.Список
    │  ├─ Навигация.Контрагент.Прочитать
    │  └─ Контрагент.Список (fallback)
    ↓
    ├─ Получение сделок через:
    │  ├─ CRMLead.getList (официальный)
    │  └─ Сделка.Список (fallback)
    ↓
    └─ Получение документов через:
       ├─ Документ.СписокДокументов
       └─ Реестр.СписокЗаписей
```

## ❓ FAQ (Часто задаваемые вопросы)

### Q: Почему я вижу "Клиент не найден в CRM"?

**A:** Возможные причины:
1. Клиента действительно нет в вашей CRM СБИС
2. Неправильные OAuth credentials
3. Недостаточно прав у OAuth приложения
4. Не подключен тариф "СБИС CRM"

### Q: Как проверить, что методы работают?

**A:** Запустите диагностику:
```bash
node backend/test-crm-integration.js
```

Или через API:
```bash
curl -X POST http://localhost:3000/api/sbis-proxy/diagnostics \
  -H "Content-Type: application/json"
```

### Q: Можно ли использовать без OAuth?

**A:** Да, но с ограничениями. Используйте обычную авторизацию:
- Эндпоинт: `/api/sbis-proxy/crm-client` (без "-oauth")
- Будут доступны только методы с правами текущего пользователя

### Q: Где взять тестовый ИНН?

**A:** Используйте ИНН из вашей CRM СБИС или:
- `7707083893` - Сбербанк (есть в публичной базе)
- Ваши реальные клиенты из CRM
