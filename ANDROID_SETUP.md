# Настройка Android приложения

## ✅ Что готово

1. **Структура проекта**
   - API клиент (Retrofit)
   - Модели данных
   - Экраны авторизации и главного экрана
   - Управление токенами

2. **Экраны**
   - LoginActivity - экран входа
   - DashboardActivity - главный экран с балансом и транзакциями

3. **Функционал**
   - Авторизация через API
   - Отображение баланса
   - Список последних транзакций
   - Быстрые действия (Услуги, История, Аналитика, Уведомления)

## 🔧 Настройка

### 1. URL Backend API

По умолчанию используется `http://10.0.2.2:3000/api/` (для Android эмулятора).

Для реального устройства или другого сервера измените в файле:
`app/src/main/java/com/example/worldcashbox/data/api/RetrofitClient.kt`

```kotlin
private const val BASE_URL = "http://ВАШ_IP:3000/api/"
```

### 2. Запуск приложения

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Запустите на эмуляторе или устройстве

### 3. Тестирование

1. Убедитесь, что backend сервер запущен на порту 3000
2. Запустите приложение
3. Войдите с существующими учетными данными

## 📱 Структура приложения

```
app/src/main/java/com/example/worldcashbox/
├── data/
│   ├── api/              # Retrofit API клиент
│   └── model/            # Модели данных
├── ui/
│   ├── login/            # Экран авторизации
│   └── dashboard/        # Главный экран
└── utils/                # Утилиты (TokenManager)
```
