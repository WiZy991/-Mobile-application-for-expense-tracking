# Установка multer

Сервер не запускается из-за отсутствия модуля `multer`. 

## Решение:

Выполните в терминале в папке `backend`:

```bash
npm install multer
```

Или если используете PowerShell:

```powershell
cd backend
npm install multer
```

После установки сервер должен запуститься автоматически (если используется nodemon) или перезапустите его вручную:

```bash
npm start
# или
npm run dev
```
