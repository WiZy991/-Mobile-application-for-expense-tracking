# 🚀 Начните здесь - Проект на Kotlin

## ✅ Миграция завершена!

Проект успешно мигрирован на Kotlin. Теперь и backend, и Android приложение используют единый язык.

## 📁 Что где находится

### ✅ Готово к использованию:

1. **`app/`** - Android приложение (Kotlin)
   - Полностью готово
   - Не требует изменений
   - Работает с новым backend

2. **`backend-kotlin/`** - Backend сервер (Kotlin/Ktor)
   - Все 14 роутов перенесены
   - Готов к использованию
   - Запускается на порту 3000

### ❌ Можно удалить:

3. **`backend/`** - Старый Node.js backend
   - Можно удалить после проверки нового backend

4. **`mobile/`** - React Native приложение
   - Можно удалить (есть нативное Android)

## 🚀 Быстрый старт

### Шаг 1: Запустить Backend

```bash
cd backend-kotlin
cp env.template .env
# Отредактируйте .env - укажите DB_PASSWORD и JWT_SECRET
./gradlew run
```

### Шаг 2: Запустить Android приложение

1. Откройте проект в Android Studio
2. Убедитесь, что PostgreSQL запущен
3. Запустите приложение

Готово! 🎉
